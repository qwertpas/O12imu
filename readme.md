## Compact IMU for RS485 bus

IMU: LSM6DSVTR

MCU: STM32L011F4U

Size: 12.5x12.5

Mounting: 4 x M2 on 12mm diameter circle 

Connection: 4pin 0.5mm pitch FFC for power and RS485 communication, optional I2C solder pads
